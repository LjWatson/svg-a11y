Case: <use> aria-labelledby <title> aria-hidden unrendered

The <svg> element defines a group containing a <title>, a <rect> and a <text> element, in a <defs> element. The title is explicitly associated to the group with an aria-labelledby attribute

. The defs element is marked aria-hidden="true"

There Should Not be anything rendered as SVG. If there is, there is a bug where the user agent is rendering content that should not be rendered
