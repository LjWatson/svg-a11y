Test case: <text>

According to the SVG Accessibility API Mappings (AAM), the <text> element should map to the group role with platform accessibility API specific mappings corresponding to the HTML <p> element.

Test example

Test criteria

  • The browser should map the <text> element to the group role. Note: depending on the accessibility API, the text may be exposed as "paragraph" or "text".
  • The screen reader should announce the content of the <text> element.
