Case: <use> locally aria-labelledby <title>

The <svg> element defines a group containing a <rect> in a <defs> element. The rectangle is then incorporated into the document with a <use> element containing a <title> explicitly associated to the <use> element with an aria-labelledby attribute

For a user agent to pass this test, it must be possible to navigate to the rectangle and discover the title, which is "Rectangle".
