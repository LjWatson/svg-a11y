Test case: <desc> + aria-describedby

According to the SVG Accessibility API Mappings (AAM), the SVG <desc> element should provide the accessible description for its parent element. This test cases uses the aria-describedby attribute to associate the <desc> element with its parent.

Test example

Test criteria

  • The browser should use the <desc> element as the accessible description for the <svg> element, based on the accessible name and description computation.
  • The screen reader should announce the content of the <desc> element as the accessible description for the <svg> element.
