Test case: <defs> + <g> + <use> + aria-describedby

The <svg> element defines a group containing a <title>, a <desc> and a <rect> in a <defs> element. The title and desc are explicitly associated to the group with aria-labelledby aria-describedby attributes, respectively. The rectangle is then incorporated into the document with a <use> element

For a user agent to pass this test, it must be possible to navigate to the rectangle and discover the title, which is "Rectangle", and to get the description, which is "A lilac rectangle".
