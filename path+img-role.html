Test case: <path> + img role

According to the SVG Accessibility API Mappings (AAM), the <path> element should implicitly map to the ARIA img role. This test case explicitly applies role="img" to the <path> element.

Test example

Test criteria

  • The browser should map the <path> element to the img role. Note: depending on the accessibility API the browser may expose the shape as either "image" or "graphic".
  • The screen reader should indicate that the shape is either a "graphic" or "image".
