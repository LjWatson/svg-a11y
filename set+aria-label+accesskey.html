Test case: <set> + aria-label + accesskey

The <set> element enables an attribute value to be set for a specified duration.

Test example

Test criteria

  • The browser should expose the accesskey on the SVG <circle> element.
  • When triggered with the accesskey g, the aria-label for the <circle> element should change to "Pass".
  • The browser should expose "A circle" as the initial accessible name for the <circle> element, then "Pass" once the accesskey has been triggered.
  • The screen reader should announce "A circle" as the accessible name for the <circle> element, then "Pass" once the accesskey has been triggered.
