Test case: <defs> + <title> + aria-labelledby

According to the SVG Accessibility API Mappings (AAM), the <defs> element should not map to any role. The SVG <title> element should provide the accessible name for its parent element. This test case uses the aria-labelledby attribute to associate the <title> element with its parent.

Test example

Test criteria

  • The browser should not expose the <defs> element.
  • The browser should not expose the <title> element.
  • The screen reader should not recognise the SVG content.
