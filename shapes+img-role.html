Test case: SVG basic shapes + ARIA img role

According to the SVG Accessibility API Mappings (AAM), each of the SVG2 basic shapes should implicitly map to the ARIA img role. This test case explicitly applies role="img" to each basic shape element.

Test cases

rect element

circle element

ellipse element

line element

polyline element

polygon element

Pass criteria

  • A browser should map the shape to the img role through the available platform accessibility APIs. Note: depending on the API the browser may expose the shape as either "image" or "graphic".
  • Ascreen reader should indicate that the shape is either a "graphic" or "image".
