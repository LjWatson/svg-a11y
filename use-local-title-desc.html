Case: <use> locally aria-labelledby <title> aria-describedby <desc>

The <svg> element defines a group containing a <rect> in a <defs> element. The rectangle is then incorporated into the document with a <use> element containing a <title>, and a <desc>.

For a user agent to pass this test, it must be possible to navigate to the rectangle and discover the title, which is "Rectangle", and to get the description, which is "A lilac rectangle".
