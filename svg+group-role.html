Test case: <svg> + group role

According to the SVG Accessibility API Mappings (AAM), the <svg> element should implicitly map to the ARIA group role. This test case explicitly applies role="group" to the <svg> element.

Test example

Test criteria

  • The browser should map the <svg> element to the group role through the available platform accessibility APIs.
  • The screen reader should not announce the <svg> element.
