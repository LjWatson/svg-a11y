Test case: <desc> + <ul>

According to the SVG Accessibility API Mappings (AAM), the SVG <desc> element should provide the accessible description for its parent element. The HTML <ul> element should not be permitted inside the <desc> element.

Test example

Test criteria

  • The browser should expose the <desc> as the accessible description for the rectangle.
  • The browser should not expose the <ul> element.
  • The screen reader should not announce the <ul> semantics.
